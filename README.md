# 👟 Dynamic Shoe Configurator & E-Commerce Showcase

A highly interactive front-end landing page and custom e-commerce product configurator built using semantic **HTML5**, **CSS Custom Variables**, and **Vanilla JavaScript**, supercharged by **GSAP (GreenSock Animation Platform)**.

This project delivers an agency-level user experience through fluid animations, state-driven interfaces, and premium micro-interactions.

---

## 🔥 Key Features

* **📦 Product Configurator Modal:** Open a dynamic detail overlay to customize shoe parameters, adjust item quantities, and swap variant styles seamlessly.
* **⚡ GSAP UI Physics:** Dynamic stagger entry timelines, interactive magnetic action buttons that react to the cursor, and 3D card tilt matrices tracking mouse physics.
* **🎨 Color Backdrop Morphing:** Selecting a different sneaker colorway triggers a fluid background circle transition that shifts perfectly to match the variant's hex theme.
* **🛒 Reactive Shopping Cart:** Real-time state array updates an interactive cart drawer, featuring a custom GSAP number counter that rolls up the subtotal like an odometer.
* **🌓 Native Dark Mode Matrix:** A clean environment toggle that shifts custom CSS variable tokens seamlessly between bright layout aesthetics and dark midnight view states.
* **🎉 Confetti Checkout Simulation:** Pressing the order button triggers an immediate canvas confetti blast celebrating user interaction conversion.

---

## 🛠️ Tech Stack & Dependencies

* **Structure & Layout:** HTML5 Semantic Markup
* **Styling Matrix:** CSS3 (Variables, Flexbox, Grid, Dynamic Transitions)
* **Animation Suite:** GSAP Core Platform (GreenSock v3) via CDN
* **Effects Integration:** Canvas-Confetti Script Engine via CDN
* **Logic Framework:** Vanilla JavaScript (ES6+ State-Linked DOM Rendering)
